System Overview & Current Challenges

  • Current MIS: PHP-based, hosted on a single DigitalOcean droplet.
  • Database: MySQL, co-hosted on the same server.
  • QuickBooks: Not yet integrated with MIS.
  • Payments: USSD-based, but disconnected from financial reporting.

The integration aims to modernize this architecture with a microservices approach and real-time QuickBooks synchronization.
